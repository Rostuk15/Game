## Description
Alien Shooter is a simple arcade-style game built with Python where the player controls a spaceship and shoots incoming aliens.

- Player spaceship movement
- Shooting mechanics
- Enemy (alien) spawning
- Collision detection
- Score tracking
- Game loop logic
- Sprite handling

## Tech Stack
